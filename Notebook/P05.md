# P05 - QIIME 2 Diversity Analysis  

February 20, 2026

## Purpose:
To quantify microbial diversity within samples (alpha diversity) and between samples (beta diversity).

## Material: 
- Filtered feature table
- Rooted phylogenetic tree
- Selected sampling depth
- Metadata file

## Method:
1. Performed core diversity metrics analysis.
2. Calculated alpha diversity indices.
3. Calculated beta diversity distances.
4. Tested statistical significance between metadata groups.

## Results: 
Alpha diversity metrics measured within-sample diversity, while beta diversity analyses revealed differences in microbial communities across metadata categories.
Principal coordinate analysis (PCoA) plots visualized clustering patterns among samples.

## Discussion:
Statistical tests identified whether microbial community composition differed significantly between metadata groups.
