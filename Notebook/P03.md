# P03 - QIIME 2 Taxonomic Analysis 

February 20, 2026

## Purpose:
To assign taxonomy to ASVs using a reference database and visualize microbial composition across samples.

## Material: 
- Representative sequences (rep-seqs.qza)
- Feature table (table.qza)
- SILVA reference classifier
- Sample metadata

## Method:
1. Used a pre-trained SILVA classifier targeting the V3–V4 region.
2. Assigned taxonomy to ASVs using a Naive Bayes classifier.
3. Generated taxonomy visualizations and barplots.
4. Filtered mitochondrial and chloroplast sequences.

## Code: 
[Script](../QIIME2%20/Script/Code.R)

## Output Files:
- [taxonomy.qzv](../QIIME2%20/Export/taxonomy.qzv)
- [taxa_barplot.qzv](../QIIME2%20/Export/taxa_barplot.qzv)
- [table.qzv](../QIIME2%20/Export/table_gc_intestinal_diffuse.qzv)
   
## Results: 
Taxonomic classification assigned ASVs to taxonomic ranks including domain, phylum, genus, and species (when possible). Barplots illustrated microbial community composition across samples.

## Discussion:
Removing mitochondrial and chloroplast reads prevented contamination from host or plant DNA, ensuring only microbial sequences were analyzed.
