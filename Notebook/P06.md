# P06 - Investigating the Microbiome Composition in Intestinal vs Diffuse Gastric Cancer and Evaluating the Influence of Biological Sex


## Purpose: 
The purpose of this analysis is to determine whether the overall gut microbial community differs between Lauren gastric cancer subtypes (intestinal vs diffuse), and to test whether any subtype-related differences depend on biological sex. Using the gastric cancer subset of samples with intestinal or diffuse histopathology (and available sex information where required), we will compare within-sample diversity (alpha diversity; e.g., Shannon and Faith’s PD) and between-sample community structure (beta diversity; Bray–Curtis and weighted/unweighted UniFrac) across subtypes, then extend the same framework to include sex and a subtype × sex interaction to evaluate sex-specific patterns in microbiome composition.
 

## Material: 
This analysis will use the QIIME2-exported gastric cancer microbiome files needed for downstream statistics in R: the feature/ASV abundance table (feature_table_gc_intestinal_diffuse.tsv), taxonomy assignments (taxonomy.tsv), and sample metadata (metadata.tsv) containing Lauren subtype (intestinal vs diffuse) and biological sex. A phylogenetic tree file will be included to compute weighted and unweighted UniFrac distances. All work will be done on a local computer.

## Method:
We loaded the QIIME2 export files (feature table, taxonomy, metadata, and tree) into R and built a phyloseq object. We used the Lauren subtype column (“Histopathology (lauren classification)”) to label samples as Intestinal or Diffuse and removed Mixed type. For Aim 1, we compared alpha diversity (Shannon) between subtypes and tested beta diversity differences using Bray–Curtis and UniFrac distances. We made PCoA plots and ran PERMANOVA (adonis2, 9,999 permutations). We also checked dispersion with betadisper before interpreting PERMANOVA. For Aim 3, we included Gender and tested whether subtype effects change by sex (Subtype * Gender) for both alpha and beta diversity.

## Results: 
Aim 1 (n = 45; Diffuse = 30, Intestinal = 15):
Shannon was higher in Intestinal using a linear model (p = 0.0457), but Wilcoxon was not significant (p = 0.236). Bray–Curtis PERMANOVA showed a subtype difference (R² = 0.0885, p = 0.0012) with borderline dispersion (betadisper p = 0.0538). Weighted UniFrac was not significant (p = 0.3127). Unweighted UniFrac showed a trend (p = 0.0757). Dispersion was fine for both UniFrac metrics.

Aim 3:
Shannon showed a strong Subtype * Gender interaction (p = 0.000217), meaning subtype differences depend on sex. PERMANOVA for Subtype * Gender was significant for Bray–Curtis (R² = 0.2649, p = 1e-04) and unweighted UniFrac (R² = 0.1032, p = 0.0368), with no dispersion issue. Weighted UniFrac was also significant (R² = 0.2308, p = 0.0063), but dispersion was significant (betadisper p = 0.0093), so that one needs caution. Intestinal + female was a small group (n = 3).
