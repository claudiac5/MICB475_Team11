# P06 - Investigating the Microbiome Composition in Intestinal vs Diffuse Gastric Cancer and Evaluating the Influence of Biological Sex

February 23, 2026

## Purpose: 
The purpose of this analysis is to test whether microbial community structure differs between Lauren gastric cancer subtypes (Intestinal vs Diffuse), and whether subtype-related patterns vary by biological sex. We compare within-sample diversity (alpha diversity; Shannon) and between-sample community structure (beta diversity; Bray–Curtis, weighted UniFrac, unweighted UniFrac) across subtypes (Aim 1), then extend the same framework to include sex and a subtype × sex interaction (Aim 3).

## Material: 
This analysis uses QIIME2-exported files for the gastric cancer subset: a feature/ASV abundance table (feature_table_gc_intestinal_diffuse.tsv), taxonomy assignments (taxonomy.tsv), sample metadata (metadata.tsv) containing Lauren subtype and sex, and a phylogenetic tree (tree.nwk) required for UniFrac distances. All analyses were performed locally in R using phyloseq.


## Method:
We imported the QIIME2-exported feature table, taxonomy table, metadata, and phylogenetic tree into R and used them to construct a phyloseq object. Taxonomy strings were split into standard ranks from Domain to Species, and taxonomic prefixes were removed. We then filtered the dataset to include only gastric cancer (GC) samples with Lauren subtype classified as Intestinal or Diffuse, excluding mixed or other histologic categories.

For all Aim 1 analyses, the filtered dataset was rarefied to an even sequencing depth of 8,386 reads per sample. Samples below this depth would have been removed prior to rarefaction, although no additional exclusions were required in the final dataset. After rarefaction, the dataset contained 45 samples total.

For Aim 1A, we assessed alpha diversity within samples using the Shannon diversity index and compared Intestinal and Diffuse subtypes using a Wilcoxon rank-sum test. To assess beta diversity, we calculated Bray-Curtis, weighted UniFrac, and unweighted UniFrac distance matrices, visualized patterns using principal coordinates analysis (PCoA), and tested for differences in community composition between subtypes using PERMANOVA with 9,999 permutations. Homogeneity of dispersion was evaluated using betadisper.

For Aim 1B, we examined sex-based differences within each Lauren subtype separately. Within Diffuse and Intestinal groups, Shannon diversity was compared between males and females using Wilcoxon rank-sum tests. Beta diversity was assessed within each subtype using Bray-Curtis, weighted UniFrac, and unweighted UniFrac distances, with group differences tested by PERMANOVA and dispersion assessed using betadisper.

For Aim 1C, we tested whether microbial diversity patterns varied by subtype, sex, and their interaction. Shannon diversity was analyzed using a two-way ANOVA with Subtype, Gender, and their interaction term. Beta diversity was analyzed using PERMANOVA on Bray-Curtis, weighted UniFrac, and unweighted UniFrac distance matrices with the model Subtype × Gender. Dispersion was assessed across the combined subtype-by-gender groups using betadisper.

## Results: 
Aim 1A
After rarefaction to 8,386 reads per sample, 45 gastric cancer samples were retained (Diffuse = 30, Intestinal = 15). Shannon alpha diversity did not differ significantly between Lauren subtypes based on the Wilcoxon rank-sum test (p = 0.218).

For beta diversity, PERMANOVA detected significant differences between Diffuse and Intestinal samples across all three distance metrics. Bray-Curtis was significant (R² = 0.112, p = 0.0012), weighted UniFrac was significant (R² = 0.151, p = 0.0059), and unweighted UniFrac was significant (R² = 0.059, p = 0.0128). Betadisper tests were not significant for Bray-Curtis (p = 0.627), weighted UniFrac (p = 0.539), or unweighted UniFrac (p = 0.899).

Aim 1B
Within the Diffuse subtype, Shannon alpha diversity differed significantly by gender (Wilcoxon p = 8.35 × 10⁻⁵). Within the Intestinal subtype, Shannon diversity did not differ significantly by gender (Wilcoxon p = 0.348).

Within the Diffuse subtype, PERMANOVA detected significant gender-associated differences for Bray-Curtis (R² = 0.326, p = 0.0001), weighted UniFrac (R² = 0.576, p = 0.0001), and unweighted UniFrac (R² = 0.126, p = 0.0012). Betadisper was also significant for Bray-Curtis (p = 0.0021), weighted UniFrac (p = 0.0271), and unweighted UniFrac (p = 0.0092).

Within the Intestinal subtype, PERMANOVA did not detect significant gender-associated differences for Bray-Curtis (R² = 0.070, p = 0.4597), weighted UniFrac (R² = 0.046, p = 0.4811), or unweighted UniFrac (R² = 0.062, p = 0.5906). Betadisper was significant for Bray-Curtis (p = 0.0394), but not for weighted UniFrac (p = 0.4848) or unweighted UniFrac (p = 0.2151).

Aim 1C
Using the full rarefied dataset, two-way ANOVA of Shannon diversity showed a significant effect of Subtype (p = 0.00495), a significant effect of Gender (p = 1.57 × 10⁻⁶), and a significant Subtype × Gender interaction (p = 0.000246).

For beta diversity, term-by-term PERMANOVA showed significant effects of Subtype, Gender, and Subtype × Gender across all three distance metrics. For Bray-Curtis, Subtype was significant (R² = 0.112, p = 0.0001), Gender was significant (R² = 0.163, p = 0.0001), and the Subtype × Gender interaction was significant (R² = 0.057, p = 0.0056). For weighted UniFrac, Subtype was significant (R² = 0.151, p = 0.0004), Gender was significant (R² = 0.301, p = 0.0001), and the interaction was significant (R² = 0.045, p = 0.0497). For unweighted UniFrac, Subtype was significant (R² = 0.059, p = 0.0084), Gender was significant (R² = 0.057, p = 0.0115), and the interaction was significant (R² = 0.042, p = 0.0368). Betadisper tests across subtype-by-gender groups were also significant for Bray-Curtis (p = 0.0011), weighted UniFrac (p = 0.0170), and unweighted UniFrac (p = 0.0333).


## Discussion:
Aim 1 suggests subtype-related differences are more consistently detected with Bray–Curtis than with phylogenetic UniFrac metrics, which may indicate that subtype differences are driven by abundance shifts that are not strongly structured across deeper phylogenetic lineages. The alpha diversity conclusion depends on the statistical test used (t-test vs Wilcoxon), consistent with a small and unbalanced Intestinal group after rarefaction.

Aim 3 indicates evidence for sex-related differences and a subtype × sex interaction in Shannon, but subgroup imbalance (Intestinal female n=3) limits stability of the interaction estimate. For beta diversity, significant dispersion across Subtype × Gender groups suggests that PERMANOVA significance may be partially influenced by differences in within-group variability; this should be noted as a limitation rather than interpreted as purely compositional separation.
