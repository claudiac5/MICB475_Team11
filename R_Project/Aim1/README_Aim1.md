# Aim 1

## What this folder contains
This folder contains all necessary files to carry out our aim 1 analysis

## How to run
1. Open the scripts
2. Make sure your working directory includes all input files
3. Run in order:
   - `AIM 1 Phyloseq.R`
   - `AIM 1 Analysis.R`

## Inputs
- `metadata.tsv`
- `feature_table_gc_intestinal_diffuse.tsv`
- `taxonomy.tsv`
- `tree.nwk`

## Outputs
Figures/tables are generated by the scripts (not tracked in this repo).
